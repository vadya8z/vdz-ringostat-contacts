== Installation ==
<p>
В административной панели:<br/>
<ol>
<li>
Плагины->Добавить новый->В поле ввода "Поиск плагинов"-> Ввести "VDZ Ringostat Contacts".
</li>
<li>
Установить найденный плагин.
</li>
<li>
Активировать плагин VDZ Ringostat Contacts
</li>
<ol>
</p>
<p>
FTP:<br/>
<ol>
<li>
Распаковать архив с плагином
</li>
<li>
Загрузить папку с плагином в директорию /wp-content/plugins/ на вашем сервере
</li>
<li>
Активировать плагин VDZ Ringostat Contacts в разделе "Плагины" административной панели WordPress
</li>
<ol>
</p>
